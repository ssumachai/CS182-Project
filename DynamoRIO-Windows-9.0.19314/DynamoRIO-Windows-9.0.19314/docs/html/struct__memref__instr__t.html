◆ encoding

unsigned char _memref_instr_t::encoding[MAX_ENCODING_LENGTH]

The instruction's raw encoding. This field is only valid when the the file type (see TRACE_MARKER_TYPE_FILETYPE) has OFFLINE_FILE_TYPE_ENCODINGS set. DynamoRIO's decode_from_copy() (or any other decoding library) can be used to decode into a higher-level instruction representation.

◆ encoding_is_new

bool _memref_instr_t::encoding_is_new

Indicates whether the encoding field is the first instance of its kind for this address. This can be used to determine when to invalidate cached decoding information. This field may be set to true on internal file divisions and not only when application code actually changed.
