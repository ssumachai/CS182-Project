Detailed Description

pt2ir_t is a class that can convert PT raw trace to DynamoRIO's IR.

Member Function Documentation

◆ convert()

pt2ir_convert_status_t pt2ir_t::convert ( OUT instrlist_autoclean_t &  ilist)

Returns pt2ir_convert_status_t. If the convertion is successful, the function returns PT2IR_CONV_SUCCESS. Otherwise, the function returns the corresponding error code.

Note
The convert function performs two processes: (1) decode the PT raw trace into libipt's IR format pt_insn; (2) convert pt_insn into the DynamoRIO's IR format instr_t and append it to ilist.
The caller does not need to initialize ilist. But if the convertion is successful, the caller needs to destroy the ilist.
