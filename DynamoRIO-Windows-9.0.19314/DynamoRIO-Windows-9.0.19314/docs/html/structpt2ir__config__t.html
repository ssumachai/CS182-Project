Detailed Description

The struct pt2ir_config_t is a collection of one PT trace's configurations. drpt2trace and raw2trace can use it to construct pt2ir_t.

Note
XXX: Multiple PT raw traces for a process will share one kcore dump file. We may need a kernel image manager to avoid loading the same dump file many times.
